# Pipeline notebooks

Run Jupyter from the **repository root** with `PYTHONPATH` set to the repo (same as CLI usage) so `import pipeline` resolves.

- `appraisal_structure_analysis.ipynb` — exploratory appraisal structure (see also `python -m pipeline.appraisal_structure`).
- `multi_model_comparison.ipynb`, `pipeline_outputs_analysis.ipynb` — analysis / comparison helpers.
